A program to simulate shopping in a grocery store using text file input and output in Python, Java, and Bash.

## Introduction

This program simulates a grocery store, giving random items along with randomized prices for them as well.
You enter your budget in and then select the items you want, as long as they fit within your budget. 
The items you bought are then stored within a text file, which will store all of the bought items from all the times the program was run.
This code is written in both Python and Java to combine increased speed with convienent programming.

## How to get Started with GrocerySimulator

In order to run this program, first clone the repository to your local machine. 
After this, run the command ```bash tool/src/bash.sh```, entering the asked for information at every prompt.
